## What I Do Best

<table>
<tr>
<td width="50%" valign="top">

### 📱 Mobile Engineering
Building production-grade Flutter apps with **GetX MVC**, Firebase, push notifications, and native platform integrations. I've shipped apps across chat systems, fitness management, media processing, and AI assistants — all cross-platform from a single codebase.

**Key work:** NFC card systems, ML Kit computer vision, real-time messaging, payment integration with Razorpay

</td>
<td width="50%" valign="top">

### 🤖 AI & Intelligent Systems
Designing multi-agent AI architectures, RAG pipelines, and LLM-powered applications. I integrate OpenAI, Claude, Gemini, and open-source models into production backends — not just API wrappers, but systems that reason, retrieve, and adapt.

**Key work:** Conversational AI platforms, hair disease detection (VGG19/CNN), document OCR & analysis, AI code generation

</td>
</tr>
<tr>
<td width="50%" valign="top">

### ⚡ Backend Architecture
Node.js and Python backends built for scale — JWT auth, job queues, WebSocket channels, Docker containerization, and nginx reverse proxies. I care about clean API design, proper error handling, and systems that don't break at 3 AM.

**Key work:** Multi-tenant build systems, real-time chat backends, async job processing, i18n support

</td>
<td width="50%" valign="top">

### 🔧 DevOps & Delivery
Docker-based build pipelines, shell scripting for automation, PM2 process management, and cloud deployments across Firebase, AWS, and Vercel. I believe in reproducible builds and one-command deploys.

**Key work:** Concurrent Docker build systems, automated deployment scripts, nginx multi-tenant hosting

</td>
</tr>
