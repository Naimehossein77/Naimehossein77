### What I work on

**Mobile apps** — Been doing Flutter since its early days. Built chat apps, AI assistants, gym management systems, media managers, and more. GetX MVC is my go-to architecture. I've also done native iOS (Swift/ObjC) and Android when the project needs it.

**Backend systems** — Node.js/Express for most production work, FastAPI when Python makes more sense, Spring Boot when Java is the requirement. Real-time chat servers, job queue systems, multi-tenant hosting, payment processing, auth systems — built them all from the ground up.

**AI/ML** — This is where things get interesting. I work with OpenAI, Claude, Gemini, and open-source models to build actual products. RAG pipelines, multi-agent systems, LLM orchestration with LangChain, and computer vision with TensorFlow/ML Kit. Built an AI app generator that takes a text prompt and outputs a working Flutter app with a live preview URL.

**DevOps** — Docker containerization, nginx reverse proxies, PM2 process management, shell scripting for deployment automation. I like my deployments boring and predictable.
